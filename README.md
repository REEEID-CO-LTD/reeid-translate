# REEID Translate

A WordPress plugin that translates posts, pages, Elementor and Gutenberg content with SEO-aware slugs and hreflang helpers.

## Features
- Elementor + Gutenberg flows
- WooCommerce product support
- Native-script or transliterated slugs (configurable)
- Hreflang/canonical helpers
- Bulk translate
- Debug logging

## Requirements
- WordPress 6.0+, PHP 7.4+
- OpenAI API key (store in **Settings → REEID Translate**)

## Install
1. Upload to `/wp-content/plugins/reeid-translate/`
2. Activate in **Plugins**
3. Set your OpenAI key in plugin settings
